```text
Repository file: [path/to/file.step]
Source project: [KiCad kicad-packages3D / KiCad kicad-packages3D-source / FreeCAD-library]
Original source URL: [URL]
Original source path: [upstream path or archive path]
Original commit/version/date: [commit hash, archive version, or access date, if known]
Original author(s): [KiCad library contributors / FreeCAD-library part author(s)]
Source license: [CC-BY-SA-4.0 WITH KiCad-libraries-exception / CC-BY-3.0]
Generated/modified by: [project or maintainer]
Modifications: [added header; renamed; normalized; converted; extended length; generated size variant; changed dimensions; etc.]
Output license / notice: [same as applicable source license]
No endorsement: Upstream source is credited for attribution only; no endorsement is implied.
```

Recommended machine-readable attribution manifest columns:

```csv
repo_path,source_project,original_source_url,original_source_path,original_commit_or_version,original_authors,source_license,derived_from,modifications,output_license,notes
```
